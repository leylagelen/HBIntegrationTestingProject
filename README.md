## HBIntegrationTestingProject
___

#### Test Senaryosu Adımları
* GET ve POST isteklerinin http statusünü kontrol et. 
* 200 ise success kabul et.

#### Gereksinimler: 
* Java8+
* Maven 3.6.1

#### Kullanılan test araçları:
* JUnit
* Rest Assured

Maven paketleri kurmak için
```
mvn clean package
```
komutu kullanılabilir.
